1. Object Syntax

Object Syntax

We can pass an object to v-bind:class to dynamically toggle classes

co-exist with the plain class

The v-bind:class directive can also co-exist with the plain class attribute.

Inline Object Syntax

The bound object on v-bind:class can be inline

Computed Property Syntax

We can also bind to a computed property that returns an object

2. Array Syntax

We can pass an array to v-bind:class to apply a list of classes

We can also toggle a class with a ternary expression in the list conditionally.

We can also use object syntax inside array syntax.
